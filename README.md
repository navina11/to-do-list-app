# To-Do List Application

This repository contains two versions of a To-Do List application:
1. A Vanilla JavaScript version
2. A React version

## Projects

### 1. JavaScript To-Do App
- Built using HTML, CSS, and JavaScript
- Features: Add, edit, delete tasks, dark mode, keyboard shortcuts
- Uses localStorage for data persistence

### 2. React To-Do App
- Built using React with hooks
- Features: CRUD operations, dark mode, responsive UI
- Demonstrates component-based architecture
